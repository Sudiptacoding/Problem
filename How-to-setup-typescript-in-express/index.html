To add TypeScript to an Express.js project, follow these steps: Initialize your project and install dependencies: First, make sure you have Node.js and npm installed. Then, create a new directory for your project, initialize a new Node project, and install Express and TypeScript. bash Copy code mkdir my-express-app cd my-express-app npm init -y npm install express npm install typescript ts-node @types/node @types/express --save-dev Initialize TypeScript: Create a tsconfig.json file to configure TypeScript. bash Copy code npx tsc --init In your tsconfig.json, you may want to set some useful configurations: json Copy code { "compilerOptions": { "target": "es6", "module": "commonjs", "outDir": "./dist", "rootDir": "./src", "strict": true, "esModuleInterop": true } } Set Up Folder Structure: Create a src folder, which will hold your TypeScript files, and an index.ts file, which will be the entry point of your Express app. bash Copy code mkdir src touch src/index.ts Write Express App in TypeScript: Inside src/index.ts, you can now write your Express code in TypeScript. typescript Copy code import express, { Request, Response } from 'express'; const app = express(); const PORT = 3000; app.get('/', (req: Request, res: Response) => { res.send('Hello, TypeScript with Express!'); }); app.listen(PORT, () => { console.log(`Server is running on http://localhost:${PORT}`); }); Add Scripts to package.json: Update the package.json to include scripts to compile and run the TypeScript files. json Copy code "scripts": { "build": "tsc", "start": "node dist/index.js", "dev": "ts-node src/index.ts" } build: Compiles the TypeScript files to JavaScript. start: Runs the compiled JavaScript files. dev: Directly runs the TypeScript files using ts-node. Run the App: To start the development server, use: bash Copy code npm run dev For a production build, first compile TypeScript and then start the server: bash Copy code npm run build npm start Your Express app is now set up with TypeScript! This setup will enable type checking and give you better development experience in TypeScript.
