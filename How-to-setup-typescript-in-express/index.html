Express.js প্রোজেক্টে TypeScript যুক্ত করতে নিচের ধাপগুলো অনুসরণ করুন: ১. প্রোজেক্ট ইনিশিয়ালাইজ এবং প্রয়োজনীয় ডিপেন্ডেন্সি ইন্সটল করুন প্রথমে নিশ্চিত করুন যে আপনার কম্পিউটারে Node.js এবং npm ইন্সটল করা আছে। এরপর প্রোজেক্টের জন্য একটি নতুন ডিরেক্টরি তৈরি করুন, প্রোজেক্টটি ইনিশিয়ালাইজ করুন এবং Express এবং TypeScript ইন্সটল করুন। mkdir my-express-app cd my-express-app npm init -y npm install express npm install typescript ts-node @types/node @types/express --save-dev ২. TypeScript ইনিশিয়ালাইজ করুন প্রোজেক্টে TypeScript কনফিগার করার জন্য একটি tsconfig.json ফাইল তৈরি করুন। npx tsc --init এরপর tsconfig.json ফাইলে কিছু প্রয়োজনীয় সেটিংস করতে পারেন: { "compilerOptions": { "target": "es6", "module": "commonjs", "outDir": "./dist", "rootDir": "./src", "strict": true, "esModuleInterop": true } } ৩. ফোল্ডার স্ট্রাকচার তৈরি করুন প্রোজেক্টে একটি src নামের ফোল্ডার তৈরি করুন, যা আপনার TypeScript ফাইলগুলো রাখবে। সেই সাথে একটি index.ts ফাইল তৈরি করুন, যা আপনার Express অ্যাপের এন্ট্রি পয়েন্ট হবে। mkdir src touch src/index.ts or src/index.ts ৪. TypeScript এ Express অ্যাপ লিখুন src/index.ts ফাইলের ভিতরে Express কোড লিখুন TypeScript ব্যবহার করে। import express, { Request, Response } from 'express'; const app = express(); const PORT = 3000; app.get('/', (req: Request, res: Response) => { res.send('Hello, TypeScript with Express!'); }); app.listen(PORT, () => { console.log(`Server is running on http://localhost:${PORT}`); }); ৫. package.json এ স্ক্রিপ্ট যুক্ত করুন package.json এ স্ক্রিপ্টস যুক্ত করুন যাতে TypeScript ফাইল কম্পাইল এবং রান করা যায়। "scripts": { "build": "tsc", "start": "node dist/index.js", "dev": "ts-node src/index.ts" } build: TypeScript ফাইলগুলোকে JavaScript এ কম্পাইল করে। start: কম্পাইল করা JavaScript ফাইল রান করে। dev: ts-node ব্যবহার করে সরাসরি TypeScript ফাইল রান করে। ৬. অ্যাপ রান করুন ডেভেলপমেন্ট সার্ভার চালানোর জন্য নিচের কমান্ডটি ব্যবহার করুন: npm run dev প্রোডাকশনে চালাতে চাইলে প্রথমে TypeScript কম্পাইল করুন এবং তারপর সার্ভার চালান: npm run build npm start এভাবেই আপনি আপনার Express অ্যাপ্লিকেশন TypeScript সহ সেটআপ করতে পারেন। TypeScript যুক্ত করার ফলে টাইপ চেকিং এবং উন্নত ডেভেলপমেন্ট অভিজ্ঞতা পাবেন।
